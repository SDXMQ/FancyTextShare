# ✨ FancyTextShare

🌎 **[English Version](./README_en.md)**

FancyTextShare는 텍스트에 몽환적이고 아름다운 감성 디자인을 입혀 링크 하나로 공유할 수 있는 **프리미엄 서버리스 텍스트 카드 제작/공유 웹 서비스**입니다. 

정적 웹 애플리케이션으로 동작하여 별도의 데이터베이스나 서버 없이 오직 생성된 **단 한 줄의 URL**만으로 상대방에게 내가 작성한 감성과 배경 셰이더 그래픽을 완벽하게 그대로 전달할 수 있습니다.

---

## 🎨 주요 특징

### 1. Cinematic WebGL Shaders (Three.js)
기존의 단순한 그라데이션이나 무거운 이미지 대신, 고성능 **Three.js 기반의 GLSL Fragment Shader** 그래픽 엔진을 탑재했습니다. 
* **밤하늘 (Starry Night)**: 은하수의 부드러운 흐름과 밤하늘 전체를 유유히 횡단하여 사선으로 떨어지는 무작위 크기/속도의 별똥별 유성우.
* **새벽 (Dawn)**: 보랏빛-붉은 노을 띠 위에 부드러운 바람을 타고 자유롭게 360도로 유랑하며 빛나는 반딧불이 입자.
* **아침 (Morning)**: 흘러가는 입체 구름 노이즈와 청량한 아침 하늘.
* **노을 (Sunset)**: 노이즈 기반의 붉은 저녁놀 구름 질감.
* **오로라 (Aurora)**: 밤하늘 아래 녹색과 파란색 빛이 춤추는 듯한 리얼한 오로라 커튼 연출.
* **심해 (Deep Sea)**: 해저 속 물결 일렁임(Caustics)과 유기적으로 찌그러지며 은은하게 떠오르는 디테일한 입체 물방울 거품.
* **현재날씨 (Live Weather)**: 위치 권한 동의 하에 Open-Meteo API를 연동하여 실제 날씨(비, 눈 입자 애니메이션) 및 현재 시간대(낮/밤/황혼)에 따라 배경이 실시간으로 매핑되는 인터랙티브 테마.

### 2. 하이브리드 오버레이 & 모바일 반응형 최적화
* **하이브리드 렌더링**: 무겁고 해상도가 깨지는 3D 텍스트 대신, 배경 시각효과는 Three.js가 처리하고 글씨와 입체적인 동작 효과는 최상위 DOM 레이어에서 연계 처리합니다.
* **모바일 반응형 UI**: CSS 미디어 쿼리를 적극 활용해, 데스크톱에서는 텍스트 메뉴로, 모바일이나 폭이 좁은 화면에서는 콤팩트한 원형 아이콘으로 자동 전환되어 타이틀이나 화면 가림 없이 깔끔한 경험을 제공합니다.
* **성능 제한**: 모바일 구형 기기 등의 과부하 및 배터리 방전을 예방하기 위해 WebGL 렌더링 해상도를 `devicePixelRatio` 1.5배 이하로 제한하여 60fps의 아주 부드러운 움직임을 제공합니다.
* **Fallback 2D 모드**: WebGL을 지원하지 않는 브라우저이거나 Three.js 라이브러리 차단 시 자동으로 조용히 Canvas 2D 그라데이션 모드로 즉시 롤백되는 예외 안전장치가 내장되어 있어 접근성이 높습니다.

### 3. 서버가 필요 없는 초단축 공유 URL 기술
* 보통 설정 상태와 텍스트를 URL에 그냥 담으면 주소가 지나치게 길어지고 한글이 깨지거나 다른 기기에서 읽을 수 없는 문제가 있습니다.
* FancyTextShare는 설정값(테마, 글꼴, 글자 크기, 정렬, 글자 발광 여부, 비네트 여부 등)을 **단 3바이트의 이진 헤더**로 압축 및 XOR 연산 처리하고 Base64URL로 난독화 인코딩하는 초경량 기술을 적용했습니다.
* 이를 통해 데이터베이스 저장 없이도 한글 텍스트 카드를 **약 50자 안팎의 아주 짧은 주소**로 생성하여 메신저 등으로 간편히 공유할 수 있습니다.

### 4. 견고한 객체 지향(OOP) 아키텍처
* **빌드 도구 없는 파일 분할 구조**: 복잡한 외부 번들러나 로컬 웹 서버 연동 없이 `js/` 디렉토리 내부의 분리된 소스 파일들을 순차적으로 로드하여, 오프라인 환경(`file://`)에서도 `index.html` 더블 클릭만으로 즉시 실행되는 강력한 접근성을 그대로 유지했습니다.
* **확장성**: 내부 코드는 철저한 분업(SRP)에 기반해 있습니다. 다국어 처리를 캡슐화한 `I18nManager`, 전략 패턴(Strategy Pattern)으로 다양한 시각 효과를 제어하는 `EntranceEffectFactory`, 화면의 라이프사이클을 통제하는 `EditorController`/`ViewController` 등 유지보수와 기능 확장이 매우 용이한 구조를 갖추고 있습니다.

## ⚙️ 커스터마이징 옵션

* **텍스트 입력**: 최대 300자의 텍스트 입력 및 글자 수 카운터 기능.
* **배경 테마**: 밤하늘, 새벽, 아침, 노을, 오로라, 심해, 현재날씨 중 선택.
* **등장 효과**:
  * `떠오르기 (Float Up)`: 부드럽게 아래서 위로 솟아오르는 3D 모션.
  * `빛나며 (Fade Glow)`: 뽀얗게 안개가 걷히며 빛과 함께 글씨가 번지는 효과.
  * `타이핑 (Typewriter)`: 타이핑을 치는 듯 한 글자씩 출력되는 모션.
* **스타일 & 타이포그래피**:
  * 글꼴 변경 (Outfit 모던, Playfair 감성, Fira Code 테크)
  * 글자 크기 실시간 슬라이더 세부 조절 (16px ~ 56px)
  * 좌/우/가운데 정렬 버튼
  * **글자 발광 효과** ON/OFF 토글
  * **배경 비네트 효과** (가장자리를 어둡게 다듬는 연출) ON/OFF 토글

## 🚀 어떻게 사용하나요?

### 카드를 만드는 사용자
1. 텍스트 입력 창에 전하고 싶은 메시지를 작성합니다. (만약 텍스트를 적지 않으면 각 배경 테마 분위기에 어울리는 어울림 문구들이 자동으로 연동됩니다.)
2. 하단의 다양한 **배경 테마**, **등장 효과**, **스타일 옵션**들을 조절하며 나만의 감성 카드를 완성합니다.
3. 만족스럽다면 **`공유 링크 생성`** 버튼을 누릅니다.
4. 클립보드에 자동으로 복사된 링크 주소를 소중한 사람에게 메신저나 SNS로 전송합니다.

### 링크를 받은 사용자
* 별도의 과정 없이 링크를 열기만 하면 내가 편집했던 셰이더 그래픽 효과와 텍스트 디자인이 실시간 애니메이션과 함께 풀스크린으로 출력됩니다.
* 하단 우측의 **`나도 만들기`** 버튼을 눌러 언제든지 나만의 예쁜 문장 카드를 새롭게 만들어 공유할 수도 있습니다.

## 📂 프로젝트 구조

```text
FancyTextShare/
├── .github/
│   └── workflows/
│       └── static.yml    # GitHub Pages Actions 자동화 파일
├── js/                   # 역할별로 모듈화된 자바스크립트 파일들
│   ├── config.js         # 설정 및 테마 목록 데이터 정의
│   ├── i18n.js           # 다국어 처리 매니저 (I18nManager)
│   ├── effects.js        # 텍스트 등장 애니메이션 효과 클래스들
│   ├── crypto.js         # 공유 해시 난독화/복호화 암호화 엔진 (CryptoEngine)
│   ├── shader.js         # Three.js 배경 셰이더 렌더러 클래스 (BackgroundShader)
│   ├── particles.js      # Three.js 파티클 효과 렌더러 클래스 (WebGLParticles)
│   ├── weather.js        # 실시간 날씨 데이터 수신 및 매핑 매니저 (WeatherManager)
│   └── main.js           # 에디터/뷰어 컨트롤러 및 DOM 초기화 진입점
├── index.html            # 시맨틱 구조 및 SEO 최적화 메인 마크업
├── README.md             # 프로젝트 소개 안내 문서
└── style.css             # 글라스모피즘 스타일 및 모던 UI CSS
```

## 🛠️ 개발자 참여 및 로컬 실행

이 프로젝트는 별도의 설치 및 빌드 과정이 전혀 필요 없습니다!

1. 저장소를 클론하거나 Zip으로 다운로드합니다.
2. `index.html` 파일을 크롬 등 브라우저로 실행하기만 하면 로컬 환경에서 즉시 감성 카드를 만드실 수 있습니다.

### 2026-06-24
* **실시간 위치 기반 "현재날씨 (Live Weather)" 테마 추가**
  - **위치 동의 팝업 흐름 개선**: 브라우저 권한을 직접 요청하기 전 웹사이트 자체 안내 모달을 띄워 사용자 거부감을 낮추고, 수락 여부를 `localStorage`에 캐싱하여 이후 재방문 시 자연스럽게 통과되도록 설계했습니다.
  - **Open-Meteo API 연동**: 별도의 API 키 발급이 불필요한 공개 날씨 API를 통해, 수신자의 위경도 좌표로 즉시 현지 날씨 코드를 조회합니다.
  - **실시간 날씨 입자 효과**: 비(Rainy)가 올 때는 빠른 낙하 입자를, 눈(Snowy)이 올 때는 흔들리며 내리는 눈송이 애니메이션을 WebGL 캔버스 상에 덧씌워 렌더링합니다.
  - **시간대별 배경 유동 매핑**: 현재 시각(`new Date().getHours()`)을 감지하여, 낮/밤/황혼에 맞는 가장 어울리는 배경 셰이더(Morning/Sunset/Dawn/Starry Night 등)가 날씨 상태와 연계되어 자동으로 표현됩니다.
  - **제작자 프리뷰 지원 및 메모리 캐싱 (WYSIWYG)**: 제작자(Editor) 모드에서도 '현재날씨' 테마 선택 시 실제 날씨에 맞게 실시간 렌더링되도록 리팩토링했습니다. 또한, 다른 테마로 이동했다가 돌아왔을 때 위치 재조회 및 API 재호출 딜레이 없이 즉각 기존 테마가 적용될 수 있도록 획득한 날씨 코드를 메모리에 캐싱(`cachedCode`) 처리했습니다.
* **코드 모듈화 및 파일 분할 리팩토링**
  - **모놀리식 구조 해제**: 단일 파일인 `app.js`를 역할과 책임에 따라 `js/` 디렉토리 아래의 8개 소스 파일(`config.js`, `i18n.js`, `effects.js`, `crypto.js`, `shader.js`, `particles.js`, `weather.js`, `main.js`)로 완전히 분할하여 가독성과 유지보수성을 극대화했습니다.
  - **로컬 실행 환경 보존**: 모듈 로더나 번들러 없이 전통적인 순차 스크립트 로드 방식을 적용하여, 복잡한 로컬 서버 설치나 빌드 과정 없이 `index.html`을 브라우저에서 직접 더블 클릭(file:// 프로토콜)하는 것만으로 즉시 작동하도록 유지했습니다.
  - **배포 및 성능**: 다중 리소스를 HTTP/2 병렬 통신으로 다운로드하므로 첫 페이지 성능 저하가 전혀 없으며, 기존 GitHub Actions 배포 자동화 프로세스 설정을 그대로 사용할 수 있습니다.
