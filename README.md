# ✨ FancyTextShare

FancyTextShare는 텍스트에 몽환적이고 아름다운 감성 디자인을 입혀 링크 하나로 공유할 수 있는 **프리미엄 서버리스 텍스트 카드 제작/공유 웹 서비스**입니다. 

정적 웹 애플리케이션으로 동작하여 별도의 데이터베이스나 서버 없이 오직 생성된 **단 한 줄의 URL**만으로 상대방에게 내가 작성한 감성과 배경 셰이더 그래픽을 완벽하게 그대로 전달할 수 있습니다.

---

## 🎨 주요 특징

### 1. Cinematic WebGL Shaders (Three.js)
기존의 단순한 그라데이션이나 무거운 이미지 대신, 고성능 **Three.js 기반의 GLSL Fragment Shader** 그래픽 엔진을 탑재했습니다. 
* **밤하늘 (Starry Night)**: 은하수의 부드러운 흐름과 밤하늘 전체를 유유히 횡단하여 사선으로 떨어지는 무작위 크기/속도의 별똥별 유성우.
* **새벽 (Dawn)**: 보랏빛-붉은 노을 띠 위에 부드러운 바람을 타고 자유롭게 360도로 유랑하며 빛나는 반딧불이 입자.
* **아침 (Morning)**: 흘러가는 입체 구름 노이즈와 청량한 아침 하늘.
* **노을 (Sunset)**: 노이즈 기반의 붉은 저녁놀 구름 질감.
* **오로라 (Aurora)**: 밤하늘 아래 녹색과 파란색 빛이 춤추는 듯한 리얼한 오로라 커튼 연출.
* **심해 (Deep Sea)**: 해저 속 물결 일렁임(Caustics)과 유기적으로 찌그러지며 은은하게 떠오르는 디테일한 입체 물방울 거품.

### 2. 하이브리드 오버레이 & 모바일 반응형 최적화
* **하이브리드 렌더링**: 무겁고 해상도가 깨지는 3D 텍스트 대신, 배경 시각효과는 Three.js가 처리하고 글씨와 입체적인 동작 효과는 최상위 DOM 레이어에서 연계 처리합니다.
* **모바일 반응형 UI**: CSS 미디어 쿼리를 적극 활용해, 데스크톱에서는 텍스트 메뉴로, 모바일이나 폭이 좁은 화면에서는 콤팩트한 원형 아이콘으로 자동 전환되어 타이틀이나 화면 가림 없이 깔끔한 경험을 제공합니다.
* **성능 제한**: 모바일 구형 기기 등의 과부하 및 배터리 방전을 예방하기 위해 WebGL 렌더링 해상도를 `devicePixelRatio` 1.5배 이하로 제한하여 60fps의 아주 부드러운 움직임을 제공합니다.
* **Fallback 2D 모드**: WebGL을 지원하지 않는 브라우저이거나 Three.js 라이브러리 차단 시 자동으로 조용히 Canvas 2D 그라데이션 모드로 즉시 롤백되는 예외 안전장치가 내장되어 있어 접근성이 높습니다.

### 3. 서버가 필요 없는 초단축 공유 URL 기술
* 보통 설정 상태와 텍스트를 URL에 그냥 담으면 주소가 지나치게 길어지고 한글이 깨지거나 다른 기기에서 읽을 수 없는 문제가 있습니다.
* FancyTextShare는 설정값(테마, 글꼴, 글자 크기, 정렬, 글자 발광 여부, 비네트 여부 등)을 **단 3바이트의 이진 헤더**로 압축 및 XOR 연산 처리하고 Base64URL로 난독화 인코딩하는 초경량 기술을 적용했습니다.
* 이를 통해 데이터베이스 저장 없이도 한글 텍스트 카드를 **약 50자 안팎의 아주 짧은 주소**로 생성하여 메신저 등으로 간편히 공유할 수 있습니다.

### 4. 견고한 객체 지향(OOP) 아키텍처
* **단일 파일 클래스 구조**: 복잡한 외부 번들러 연동 없이 하나의 `app.js` 파일만 사용하여 오프라인 환경(`file://`)에서도 즉시 실행되는 강력한 접근성을 보장합니다.
* **확장성**: 내부 코드는 철저한 분업(SRP)에 기반해 있습니다. 다국어 처리를 캡슐화한 `I18nManager`, 전략 패턴(Strategy Pattern)으로 다양한 시각 효과를 제어하는 `EntranceEffectManager`, 화면의 라이프사이클을 통제하는 `EditorController` 등 유지보수와 기능 확장이 매우 용이한 구조를 갖추고 있습니다.

## ⚙️ 커스터마이징 옵션

* **텍스트 입력**: 최대 300자의 텍스트 입력 및 글자 수 카운터 기능.
* **배경 테마**: 밤하늘, 새벽, 아침, 노을, 오로라, 심해 중 선택.
* **등장 효과**:
  * `떠오르기 (Float Up)`: 부드럽게 아래서 위로 솟아오르는 3D 모션.
  * `빛나며 (Fade Glow)`: 뽀얗게 안개가 걷히며 빛과 함께 글씨가 번지는 효과.
  * `타이핑 (Typewriter)`: 타이핑을 치는 듯 한 글자씩 출력되는 모션.
* **스타일 & 타이포그래피**:
  * 글꼴 변경 (Outfit 모던, Playfair 감성, Fira Code 테크)
  * 글자 크기 실시간 슬라이더 세부 조절 (16px ~ 56px)
  * 좌/우/가운데 정렬 버튼
  * **글자 발광 효과** ON/OFF 토글
  * **배경 비네트 효과** (가장자리를 어둡게 다듬는 연출) ON/OFF 토글

## 🚀 어떻게 사용하나요?

### 카드를 만드는 사용자
1. 텍스트 입력 창에 전하고 싶은 메시지를 작성합니다. (만약 텍스트를 적지 않으면 각 배경 테마 분위기에 어울리는 어울림 문구들이 자동으로 연동됩니다.)
2. 하단의 다양한 **배경 테마**, **등장 효과**, **스타일 옵션**들을 조절하며 나만의 감성 카드를 완성합니다.
3. 만족스럽다면 **`공유 링크 생성`** 버튼을 누릅니다.
4. 클립보드에 자동으로 복사된 링크 주소를 소중한 사람에게 메신저나 SNS로 전송합니다.

### 링크를 받은 사용자
* 별도의 과정 없이 링크를 열기만 하면 내가 편집했던 셰이더 그래픽 효과와 텍스트 디자인이 실시간 애니메이션과 함께 풀스크린으로 출력됩니다.
* 하단 우측의 **`나도 만들기`** 버튼을 눌러 언제든지 나만의 예쁜 문장 카드를 새롭게 만들어 공유할 수도 있습니다.
